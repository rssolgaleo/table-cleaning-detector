# Прототип системы детекции уборки столиков по видео

## Описание

Система анализирует видеопоток из зала ресторана и отслеживает состояние выбранного столика:
- **EMPTY** (пустой) — в зоне столика нет людей
- **APPROACH** (подход) — человек появился у пустого столика
- **OCCUPIED** (занят) — столик занят гостем (человек находится в зоне более 5 секунд)

Для детекции людей используется модель **YOLOv8n** (Ultralytics). Зона столика задаётся вручную через `cv2.selectROI`.

## Установка и запуск

```bash
# Создание виртуального окружения
python3 -m venv venv        # macOS/Linux
python -m venv venv         # Windows

# Активация
source venv/bin/activate    # macOS/Linux
venv\Scripts\activate       # Windows

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python main.py --video videos/1.mp4
```

При запуске откроется окно — выделите зону столика мышкой и нажмите Enter.

## Выбранное видео и столик

Использовалось **Видео 2**. Выбран столик в центре кадра. Зона выделена так, чтобы захватывать стол и сиденья, но не задевать соседние столики.

## Логика детекции событий

1. **YOLOv8n** детектирует людей на каждом кадре (класс `person`, порог уверенности 0.35)
2. Проверяется пересечение bbox людей с зоной столика (ROI)
3. Переходы между состояниями:
   - `EMPTY → APPROACH`: человек появился в зоне
   - `APPROACH → OCCUPIED`: человек непрерывно в зоне более 5 секунд
   - `OCCUPIED/APPROACH → EMPTY`: человек отсутствует более 3 секунд
4. Пороги адаптируются к FPS видео автоматически

## Полученный результат

- Видео: **Видео 2** (15 FPS, 956 сек)
- Среднее время ожидания (empty → approach): **46.25 сек**
- Всего переходов empty → approach: **7**
- Всего событий: **17** (8 empty, 7 approach, 2 occupied)
